    Simple Model

    This page demonstrates how to run a simple linear macroeconomic model in DynareJulia using a .mod file and a Julia driver script. The model is a small, FPAS-style gap model with an endogenous credibility block, solved under perfect foresight.

    Download the Files

    Download both files and place them in the same project directory:

    • simple_linear.mod — the model file
    • simple_linear.jl — the Julia script that runs the model and plots the results

model;

Y = BETA1*Y(-1) + BETA2*Y(+1) - BETA3*RR_GAP(-1) + E_Y;

PIE = LAMBDA1*PIE_EXP + (1 - LAMBDA1)*PIE4(-1) + LAMBDA2*Y(-1) + E_PIE;

PIE_EXP = C*PIE4(+4) + (1 - C)*PIE4(-1) + 0.1*(1 - C) + E_PIE_EXP; 

C = 0.04*SIGNAL_CBP + (1-0.04)*C(-1) + E_C;

SIGNAL_CBP =  RES_PIE4_HIGH^2/(RES_PIE4_HIGH^2+RES_PIE4_LOW^2) ; 

// RES_PIE4_LOW =    Actual           -          Forecast
RES_PIE4_LOW =       PIE4    -      (   0.5*PIE_SS+0.5*PIE4(-1))   ;

// RES_PIE4_HIGH =    Actual           -          Forecast
RES_PIE4_HIGH =       PIE4    -      (   0.1*10 + 0.9*PIE4(-1))   ;

PIE4 = 0.25*(PIE+PIE(-1)+PIE(-2)+PIE(-3));

DL_GDP = DL_GDP_BAR + (Y - Y(-1));

DL_GDP_BAR = ALPHA1*DL_GDP_BAR(-1) + (1-ALPHA1)*G_GDP_BAR + E_DL_GDP_BAR;

G_GDP_BAR = ALPHA2*G_GDP_BAR(-1) + (1-ALPHA2)*GROWTH_SS + E_G_GDP_BAR;

RR = RS-PIE(+1);

RR_BAR = ALPHA3*RR_BAR(-1) + (1-ALPHA3)*RR_BAR_STAR + E_RR_BAR;

RR_GAP = RR - RR_BAR;

UNR_GAP = UNR_BAR - UNR;

UNR_BAR = ALPHA8*UNR_BAR(-1) + (1-ALPHA8)*UNR_SS + g_UNR + E_UNR_BAR;

g_UNR = (1-ALPHA4)*g_UNR(-1) + E_UNR_g;

UNR_GAP = ALPHA5*UNR_GAP(-1) + ALPHA6*Y + ALPHA7*Y(+1) + E_UNR;

RS = GAMMA1*RS(-1) + (1 - GAMMA1)*(RR_BAR + PIE4(+3) + GAMMA2*(PIE4(+3) - PIE_SS) + GAMMA4*Y) + E_RS;

end;

    Description:

    • The model combines an output-gap (IS) equation, a Phillips curve, an endogenous credibility block (C, SIGNAL_CBP, PIE_EXP), unemployment/NAIRU dynamics, potential-GDP growth, and a forward-looking policy rule for the nominal rate RS.
    • A steady_state_model block provides the analytical steady state, and histval sets the initial conditions (2025Q1).
    • The scenario! statements impose a sequence of inflation (E_PIE) and output-gap (E_Y) shocks over the first periods.
    • The model is solved under perfect foresight with the lmmcp (mixed complementarity) solver via perfect_foresight_setup / perfect_foresight_solver.

    ▶️ Running the Model in Julia (simple_linear.jl)

    The driver script below activates a project environment, runs the model with the @dynare macro, extracts the simulation into a DataFrame, and plots key variables (RS, Y, PIE, UNR) on a quarterly time axis.

    using Pkg
Pkg.activate("venv")

using Dynare
using PATHSolver
using DataFrames
using Dates
using Plots

# --- Set backend ---
gr()

# --- Step 1: Define Model Files ---
model_files = [
    "simple_linear.mod",
]

# --- Step 2: Run Dynare Models ---
results = Dict{String, Any}()
for model in model_files
    if isfile(model)
        println("Running model: ", model)
        results[model] = eval(Meta.parse("@dynare \"$model\""))
        println("Finished: ", model, "\n")
    else
        println("Model file not found: ", model)
    end
end

# --- Step 3: Extract Data into DataFrames ---
df_models = Dict()
for model in keys(results)
    df_models[model] = DataFrame(results[model].results.model_results[1].simulations[1].data)
end

# --- Step 4: Generate Quarterly Time Index (2025Q1–2035Q2) ---
time_index = (Date(2025,2):Quarter(1):Date(2035,3))[1:40]
formatted_labels = [string(year(d)) * "-Q" * string(month(d) ÷ 3 + 1) for d in time_index]

# --- Step 5: Plot Variables and Colors ---
variables_to_plot = [:RS, :Y, :PIE, :UNR]
model_colors = Dict(
    "simple_linear.mod" => :blue
)

# --- Step 6: Create 2x2 Grid Plot (No Legends) ---
plot_grid = plot(layout=(2,2), size=(900,700), legend=false)

for (i, var) in enumerate(variables_to_plot)
    for (model, df) in df_models
        if haskey(model_colors, model)
            y = df[!, var][1:40]  # Use the full 40 quarters, starting from 2025Q1
            plot!(1:40, y,
                  label="", lw=2, color=model_colors[model],
                  subplot=i, xrotation=90)
        end
    end
    title!(subplot=i, string(var))
    xticks!(subplot=i, 1:4:40, formatted_labels[1:4:40])
end

# --- Step 7: Save and Display ---
savefig(plot_grid, "simple_linear.png")
display(plot_grid)

    This will:

    • Activate the venv project environment and load the required packages.
    • Run simple_linear.mod through the @dynare macro, which parses, solves, and simulates the model.
    • Collect the simulated paths into a DataFrame.
    • Produce a 2×2 grid plot of the nominal interest rate (RS), output gap (Y), inflation (PIE), and unemployment rate (UNR), saved as simple_linear.png.

    !!! note Make sure simple_linear.mod and simple_linear.jl are in the same directory, and run the script from that directory so the model file is found.
